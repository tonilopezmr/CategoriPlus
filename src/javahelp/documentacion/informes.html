Informes

1.1 Generacion de informes

Actualmente se pueden generar 3 informes:

  • 1.1.1 Informe de todas las categorias.
  • 1.1.2 Informe de las fichas de produto.
  • 1.1.3 Informe final

1.1.1 Informe de categorias

El informe muestra todas las categorias que hay cargadas en el programa, las extrae de la base de datos

1.1.2 Informe de fichas de productos

El informe muestra las fichas de todos los productos que hay cargados en el programa, extrae los datos de la base de datos.

1.1.3 Informe final

El informe final muestra por cada categoria la media del precio de venta de susproductos, y todos sus productos, al final realiza la cuenta de cuantos productos hay para esa categoria.

Al final del documento hay una grafica de pastel mostrando el numero de productos de cada categoria.
